A business rule runs on server before or after a record is loaded, inserted, deleted or updated.<br><br><b>Explanation:</b> Client scripts run JavaScript on the client (web browser) when client-based events occur, such as when a form loads, a field changes value, or a form is submitted. A Business Rule is a server-side script that runs when a record is displayed, inserted, updated, deleted, or when a table is queried.
